# Atalhos

| Comando    | Função                           |
| ---------- | -------------------------------- |
| `ctrl + b` | Alterna o explorador de arquivos |
| `ctrl + o` | Abre o Telescope                 |
| `ctrl + t` | Aterna visão de terminal         |

# Capacidades

1. Sugestão e Autocomplete (cmp.nvim)
2. Explorador de arquivos (NvimTree)
3. Finder de arquivos (Telescope.nvim)
4. Gerenciador de plugins (Lazy.nvim)

# LSPs Configuradas

1. lua-language-server
2. Rust Analyzer
3. Godot
4. bash-language-server
