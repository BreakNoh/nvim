# Atalhos

| Comando    | Modo   | Função                           |
| ---------- | ------ | -------------------------------- |
| `ctrl + b` | Normal | Alterna o explorador de arquivos |
| `ctrl + o` | Normal | Abre o Telescope                 |
| `F2`       | Normal | Renomear                         |
| `gD`       | Normal | Ir pra definição                 |

# Capacidades

1. Sugestão e Autocomplete (cmp.nvim)
2. Explorador de arquivos (NvimTree)
3. Finder de arquivos (Telescope.nvim)
4. Gerenciador de plugins (Lazy.nvim)

# LSPs Configuradas

1. lua-language-server
2. Rust Analyzer
3. Godot
4. bash-language-server
